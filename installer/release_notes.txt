## 변경사항

### v1.0.26 — 모달에서 다시 다운로드 & 대기열 개선

#### 새 기능
- **다시 다운로드 버튼**: 기록 카드 상세 모달에 "다시 다운로드" 버튼 추가 — 이전에 받은 영상을 클릭 한 번으로 재다운로드 요청
- **대기열 요청자 표시**: 대기 중인 항목에 요청자(Discord 사용자명 또는 dashboard) 표시
- **대기열 URL 간략화**: YouTube URL → `youtu.be/영상ID` 형태로 단축 표시, 긴 URL 자동 줄임

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.0.26.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
