## 변경사항

### v1.1.2 — 대시보드 편의성 개선

#### 대시보드 UX 개선
- **"📂 파일 위치 보기" 버튼 추가**: 기록 모달에서 다운로드된 파일의 위치를 Windows 탐색기에서 직접 열 수 있음 — 파일이 선택된 상태로 폴더가 열림
- **탭 타이틀에 다운로드 현황 표시**: 다운로드 중일 때 브라우저 탭/창 타이틀이 "(2↓) StreamSaver"로 변경되어 다른 탭에 있어도 진행 상황 확인 가능
- **파일이 없을 경우 폴더 자동 대체**: 파일이 삭제되었어도 다운로드 폴더를 열어줌

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.1.2.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
