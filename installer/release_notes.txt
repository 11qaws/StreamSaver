## 변경사항

### v1.1.1 — 자동 업데이트 UX 개선 · 대시보드 정확도 수정

#### 자동 업데이트 전면 개선
- **백그라운드 자동 다운로드**: 새 버전 발견 시 사용자가 클릭하지 않아도 인스톨러를 자동으로 백그라운드 다운로드
- **설치 확인 다이얼로그**: 다운로드 완료 후 트레이에서 "✅ 설치 준비됨 — 지금 설치" 클릭 시 릴리즈 노트를 보여주는 확인 창 표시 — 실수로 설치되지 않음
- **시작 시 지연 체크**: 앱 시작 30초 후 업데이트 체크 (앱 시작 부하 분산)
- **3단계 트레이 메뉴**: ⬇️ 다운로드 중 (45%) → ✅ 설치 준비됨 — 지금 설치 → 실패 시 재시도 버튼
- **이중 다운로드 방지**: Lock으로 동시에 두 번 다운로드되는 경쟁 조건 제거
- **설치 다이얼로그 중복 방지**: 트레이를 여러 번 클릭해도 창이 하나만 열림
- **릴리즈 노트 전문 표시**: 기존에 200자로 잘리던 릴리즈 노트를 전체 표시

#### 대시보드 정확도 수정
- **통계·차트·필터가 첫 20개만 반영하던 버그 수정**: 기록 불러오기가 기본 20개 제한이었던 문제를 수정해 최대 5,000개 전체 기록 기준으로 통계 계산
- **"오늘 다운로드" 타임존 버그 수정**: `toISOString()`이 UTC를 반환해 한국에서 자정~오전 9시 사이에 오늘 다운로드 수가 0으로 표시되던 문제 수정 (로컬 시간 기준으로 변경)
- **검색·채널 필터가 최근 20개에만 적용되던 문제 해결**: 이제 전체 기록에서 검색 가능

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.1.1.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
