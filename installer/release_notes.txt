## 변경사항

### v1.0.21 — 대기열 표시 & 설정 정보 탭

#### 새 기능
- **대기열 실시간 표시**: 활성 다운로드 패널 하단에 대기 중인 항목 목록 표시 — 순서, URL 확인 가능
- **설정 정보 탭**: 대시보드 상단 ⚙️ 탭에서 앱 버전, 최대 동시 다운로드 수, 채널 감시 주기, 디스크 사용량, 다운로드 폴더 경로 한눈에 확인
- **디스크 사용량 바**: 설정 탭에서 시각적 사용량 바로 남은 공간 즉시 파악
- **정보 동기화 강화**: 상태 표시줄 · 활성 패널 · 대기열이 단일 `/api/status` 폴링(5초)으로 항상 동기화

#### 개선
- 설정 API(`/api/settings`) 응답에 `watch_poll_interval`, `repo` 필드 추가
- 취소 버튼 클릭 시 즉시 상태 갱신

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.0.21.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
