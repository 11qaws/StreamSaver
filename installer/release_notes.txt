## 변경사항

### v1.0.30 — 기록 항목 삭제

#### 새 기능
- **기록 항목 삭제**: 카드 상세 모달 하단 "기록 삭제" 버튼 — 특정 다운로드 기록을 선택적으로 제거
- **`DELETE /api/history?id=<영상ID>` 엔드포인트**: 외부 도구에서도 개별 기록 삭제 가능
- 삭제 확인 다이얼로그 + 토스트 피드백 + 기록 목록 즉시 갱신

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.0.30.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
