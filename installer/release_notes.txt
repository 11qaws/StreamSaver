## 변경사항

### v1.0.31 — 오늘 다운로드 통계 & 다운로드 일시 표시

#### 새 기능
- **오늘 다운로드 통계**: 기록 탭 상단 통계 요약에 "오늘 다운로드" 수 추가 — 오늘 얼마나 받았는지 한눈에 확인
- **다운로드 일시 카드 표시**: 기록 카드 하단에 다운로드한 날짜(YYYY-MM-DD) 표시 (↓ 아이콘)
- **모달 다운로드 일시**: 카드 상세 모달에 "다운로드 일시" 행 추가 — 정확한 시간 확인

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.0.31.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
