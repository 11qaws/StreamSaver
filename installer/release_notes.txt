## 변경사항

### v1.1.0 — 안정성·보안 강화 릴리즈

#### 버그 수정
- **"파일 없는 항목 정리" 실제로 동작하지 않던 버그 수정**: 기록 항목에 `file_path` 필드를 추가하고, 정리 로직이 올바른 필드를 확인하도록 수정
- **동시 다운로드 완료 시 기록 손실 방지**: 기록 파일 쓰기에 Lock + 원자적 쓰기(tmp→replace) 적용 — MAX_PARALLEL=5 환경에서 동시 완료 시 기록이 덮어써지던 문제 해결
- **정보 수집(GETTING_INFO) 중 취소 버튼이 무시되던 문제 수정**: 이제 정보 수집 단계에서도 즉시 취소 가능
- **쿠키 파일 손상 방지**: 쿠키 저장 시 원자적 쓰기 적용 — 앱 크래시 중 쓰기 발생 시 파일 깨짐 방지
- **Edge CDP 타임아웃 판정 오류 수정**: `start_headless` / `start_visible`이 CDP 응답 없을 때도 성공으로 반환하던 문제 수정

#### 보안 강화
- **웹 서버 로컬호스트 전용 바인딩**: `0.0.0.0:8080` → `127.0.0.1:8080` — 같은 WiFi의 다른 사람이 대시보드에 접근하거나 다운로드를 추가할 수 없음
- **요청 본문 크기 제한**: POST 요청 1 MB 상한 추가 — 대용량 요청으로 메모리 고갈 방지
- **API 입력값 검증 강화**: 잘못된 `task_id`·`per_page`·`page` 값에 대해 500 대신 적절한 400 반환

#### 내부 개선
- **불필요한 GitHub 동기화 코드 제거**: 다운로드 완료 후 실행되던 `git push` 코드는 설치 환경에서 동작하지 않으면서 슬롯을 낭비하던 dead code — 완전 제거
- **unarchived 관리 창 중복 열기 방지**: 트레이 메뉴에서 여러 번 클릭해도 창이 하나만 열리도록 수정

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.1.0.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
