## 변경사항

### 자동 업데이트 버그 수정
- 트레이 메뉴에 업데이트 버튼이 표시되지 않던 문제 수정 (`update_menu()` 호출 누락)
- "지금 설치" 클릭 후 반응 없던 문제 수정 (내부 정리 과정에서 hang 발생)
- 신버전 감지 → 백그라운드 다운로드 → 설치 버튼 → 클릭 시 무음 설치 → 자동 재기동 플로우 완성
- 대시보드 JS가 신버전 감지 시 서버 측 다운로드도 함께 트리거 (이중 감지 경로)

### 기타
- URL 일괄 추가 모달 textarea 비활성화 버그 수정

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.2.4.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
