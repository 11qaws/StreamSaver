## 변경사항

### v1.0.28 — 오프라인 표시 & 수동 새로고침

#### 새 기능
- **오프라인 표시**: 서버 연결이 끊기면 상태 표시줄 상단에 ⚠ 경고 표시 + 붉은 배경
- **수동 새로고침 버튼**: 상태 표시줄 우측에 ↻ 버튼 추가 — 클릭 시 상태 및 기록 즉시 새로고침 (5초 폴링 안 기다려도 됨)
- 2회 연속 연결 실패 시 오프라인으로 전환, 성공 시 즉시 복구

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.0.28.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
