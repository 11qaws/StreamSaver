## 주요 변경사항

### 대시보드 상태 표시줄 개편
- 상태 점 4개로 재구성: **서버 / Discord / 컴퓨터 / 로그인**
- 서버(릴레이 연결), Discord(봇 Discord 연결), 컴퓨터(트레이 앱 동작), 로그인(YouTube 쿠키 유효성) 각각 독립적으로 표시

### 버그 수정
- **URL 미리보기 정렬**: 좌측 정렬되던 영상 정보 미리보기를 입력창과 동일한 중앙 정렬로 수정
- **일괄 추가 모달 버튼**: 모달 열릴 때 버튼이 눌리지 않던 버그 수정
- **다운로드 중 취소**: 정보 수집 단계에서도 취소 버튼이 즉시 동작하도록 수정
- **파일 크기 0 표시**: 다운로드 완료 후 실제 파일 크기 정확히 표시

### 릴레이 서버 안정성 개선
- **Discord 봇 프로세스 분리**: Discord 봇과 WebSocket 허브를 완전히 별도 프로세스로 분리
- Discord 재연결 이슈가 발생해도 WebSocket 허브가 영향받지 않음 — 서버 동결 문제 근본 해결
- 두 프로세스는 localhost IPC TCP로 통신, 각각 systemd가 독립적으로 관리(자동 재시작)

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.2.0.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
