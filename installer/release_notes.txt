## 변경사항

### v1.1.3 — 기간 필터 · 통계 카드 인터랙션 추가

#### 대시보드 UX 개선
- **기간 필터 추가**: 다운로드 기록을 "오늘 / 최근 7일 / 이번 달 / 전체 기간"으로 필터링 가능 — 필터 바 정렬 옆에 새 드롭다운 추가
- **"오늘 다운로드" 카드 클릭 → 즉시 오늘 필터 적용**: 통계 요약의 "오늘 다운로드" 카드를 클릭하면 오늘 필터가 바로 적용되고, 다시 클릭하면 해제 — 카드 강조 표시로 현재 상태 확인 가능

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.1.3.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
