## 변경사항

### v1.0.20 — 대시보드 직접 다운로드

#### 새 기능
- **URL 직접 다운로드**: 대시보드 상단 URL 입력바에서 YouTube URL을 입력하면 Discord 없이 바로 다운로드 시작 — Enter 또는 버튼 클릭, YouTube URL 붙여넣기 자동 감지
- **토스트 알림**: 다운로드 추가, 채널 추가/삭제 등 모든 액션 결과를 우측 하단 토스트로 즉시 피드백 (3초 자동 사라짐)
- **키보드 단축키**: `D` — URL 입력란 포커스, `/` — 검색 포커스, `Esc` — 모달 닫기
- **`POST /api/download` 엔드포인트**: 외부 도구에서도 API로 다운로드 추가 가능

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.0.20.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
