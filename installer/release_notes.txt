## 변경사항

### v1.0.17 — 대시보드 제어 기능 추가

#### 새 기능
- **대시보드 탭 구조**: 다운로드 기록 탭과 채널 관리 탭으로 분리, 봇 재시작 없이 웹에서 바로 전환 가능
- **채널 관리 패널**: 대시보드에서 직접 YouTube 채널 추가/삭제 — URL 입력, 표시 이름, 제목 필터 설정 즉시 반영
- **다운로드 취소 버튼**: 진행 중인 각 다운로드 옆 취소 버튼 — 클릭 시 즉시 취소 후 패널 갱신
- **취소/채널 관리 API**: `POST /api/cancel`, `GET/POST/DELETE /api/channels` 엔드포인트 추가

#### 정보 동기화 개선
- 채널 추가/삭제 시 채널 목록 즉시 갱신
- 취소 후 상태 패널 즉시 새로고침
- 채널 관리 탭 전환 시 최신 목록 자동 로드

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.0.17.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
