## 변경사항

### 아이콘 색상 통일
- 트레이 대기 상태 아이콘: 녹색 → 청색(#2563EB) — 대시보드 색상과 통일
- 앱 아이콘(.ico): 녹색 → 청색

### 대시보드
- 탭 파비콘 추가 — 브라우저 탭에 StreamSaver 아이콘 표시
- URL 미리보기 중앙 정렬 수정
- 일괄 추가 모달 버튼 응답 버그 수정

### 업데이트 설치 개선
- 업데이트 다운로드 완료 후 "지금 설치" 클릭 한 번으로 즉시 설치
- 기존 확인 다이얼로그 제거 — 인스톨러가 백그라운드에서 무음 실행됨

### 릴레이 서버 안정성
- Discord 봇과 WebSocket 허브를 별도 프로세스로 분리 (v1.2.0 이어서)
- IPC 확장 옵션(1:N, Redis) 주석으로 문서화

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.2.1.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
