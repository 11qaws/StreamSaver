## 변경사항

### v1.1.10 — 설정 탭 업데이트 확인 버튼 추가

#### 대시보드 UX 개선
- **수동 업데이트 확인**: 설정 탭 버전 항목 옆에 "업데이트 확인" 버튼 추가 — 클릭 시 GitHub에서 최신 버전을 즉시 확인하고 결과를 토스트로 표시 (최신 버전이면 "최신 버전입니다", 새 버전이 있으면 업데이트 배너 표시)

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.1.10.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
