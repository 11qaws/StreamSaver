## 변경사항

### v1.1.24 — URL 영상 프리뷰 & 컴팩트 뷰 & 검색 UX 개선

#### 새 기능
- **URL 영상 프리뷰**: YouTube URL을 입력란에 붙여넣으면 0.6초 후 자동으로 영상 제목·채널·길이·썸네일을 미리 표시. 이미 다운로드된 영상이면 경고 메시지도 함께 표시됨
- **컴팩트 뷰 토글 (☰/⊞)**: 인포바 우측 버튼으로 카드 뷰와 컴팩트 리스트 뷰를 전환. 선택이 저장되어 다음 방문에도 유지됨
- **필터 결과 파일 크기 합계**: 결과 인포바에 "N개 중 M개 표시 · 총 X.X GB"로 필터된 항목의 전체 크기 표시
- **빈 검색 결과 개선**: 결과가 없을 때 어떤 필터가 적용되어 있는지 표시하고 '필터 초기화' 버튼 제공
- **페이지네이션 «/» 버튼**: 첫 페이지·마지막 페이지 바로 이동 버튼 추가

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.1.24.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
