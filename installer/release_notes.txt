## 변경사항

### 자동 업데이트 플로우 완성
- 신버전 감지 → 백그라운드 자동 다운로드 → 설치 버튼 표시 → 클릭 시 무음 설치 → 자동 재기동
- Python 업데이트 체커 실패 시 대시보드 JS가 직접 서버 측 다운로드를 트리거하는 이중 경로 추가
- 설치 마법사 없음 — `/VERYSILENT /NORESTART` 무음 설치

### 버그 수정
- URL 일괄 추가 모달: 제출 도중 닫으면 다음 오픈 시 텍스트박스 비활성화 상태로 남는 버그 수정

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.2.3.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
