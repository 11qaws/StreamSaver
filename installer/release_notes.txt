## 변경사항

### v1.1.25 — URL 일괄 추가

#### 새 기능
- **URL 일괄 추가 ("+ 여러 개" 버튼)**: 다운로드 버튼 옆 "여러 개" 버튼을 누르면 모달이 열리고, YouTube URL을 줄당 하나씩 붙여넣으면 감지된 URL 수를 실시간으로 확인한 후 한 번에 큐에 추가. 추가 중 진행률 바가 표시되고 완료 시 토스트로 결과 안내

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.1.25.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
