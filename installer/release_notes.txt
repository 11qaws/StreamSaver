## 변경사항

### v1.0.22 — 빠른 채널 필터 & URL 복사

#### 새 기능
- **채널 필터 칩**: 기록 탭 검색창 아래에 상위 8개 채널 버튼 표시 — 클릭 한 번으로 채널별 필터링, 활성 칩은 강조색으로 표시
- **URL 복사 버튼**: 카드 상세 모달에 "URL 복사" 버튼 추가 — 클립보드에 YouTube 링크 즉시 복사, 토스트로 확인

#### 개선
- 채널 필터 드롭다운 변경 시 칩 상태도 동기화 업데이트
- 모달 액션 버튼 두 개로 확장 (YouTube 열기 + URL 복사)

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.0.22.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
