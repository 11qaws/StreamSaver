## 변경사항

### 버그 수정
- URL 일괄 추가 모달에서 ✕·취소 버튼이 동작하지 않던 문제 수정 (메인 다운로드 상세 모달이 화면 전체를 항상 덮고 있어 클릭 차단)
- 대시보드 하단 현재 버전 표시 추가

### v1.2.4 변경사항 (이전 빌드 오류로 누락분 포함)
- 트레이 메뉴 업데이트 버튼이 표시되지 않던 문제 수정
- "지금 설치" 클릭 후 반응 없던 문제 수정
- 신버전 감지 → 백그라운드 다운로드 → 설치 버튼 → 무음 설치 → 자동 재기동 플로우 완성

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.2.5.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
