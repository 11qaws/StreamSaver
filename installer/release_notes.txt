## 변경사항

### v1.0.16 — 대시보드 개선

#### 새 기능
- **라이브 상태 표시줄**: 봇 온라인/오프라인, 쿠키 유효 여부, 디스크 여유공간, 활성 다운로드 수, 대기 수 — 5초마다 자동 갱신
- **활성 다운로드 패널**: 다운로드 진행 중일 때 진행률, 속도, 남은 시간 실시간 표시 (완료 시 자동으로 숨김)
- **정보 동기화**: 단일 상태 폴링으로 표시줄·진행 패널·기록 목록이 항상 일치, 다운로드 완료 시 기록 즉시 자동 갱신
- **전체 한국어 UI**: 검색 입력, 정렬 옵션, 채널 필터, 배지, 통계, 안내 문구 모두 한국어
- **카드 클릭 → YouTube 이동**: 기록 카드를 클릭하면 해당 YouTube 영상으로 바로 이동

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.0.16.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
