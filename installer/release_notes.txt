## 변경사항

### v1.1.26 — 일괄 추가 URL 미리보기

#### 개선
- **일괄 추가 모달 URL 미리보기**: 텍스트를 붙여넣으면 감지된 URL을 목록으로 표시. 어떤 URL이 추출되었는지 확인 후 제출 가능
- **혼합 텍스트 지원**: URL만 붙여넣지 않아도 됨. 유튜브 페이지나 메시지를 통째로 붙여넣으면 YouTube URL만 자동 추출
- **trailing 문장부호 제거**: URL 끝에 붙은 마침표·괄호·따옴표 등을 자동으로 제거

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.1.26.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
