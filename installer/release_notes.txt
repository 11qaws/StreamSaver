## 변경사항

### v1.1.4 — 다운로드 완료 알림 · 버그 수정

#### 대시보드 UX 개선
- **개별 완료 즉시 기록 업데이트**: 여러 영상을 동시에 다운로드할 때 각 영상이 완료되는 즉시 기록 목록이 갱신됩니다 — 기존에는 모든 다운로드가 끝나야 목록이 업데이트되었습니다
- **완료 토스트 알림**: 다운로드 완료 시 화면 우측 하단에 "다운로드 완료 (+N개)" 토스트 알림 표시

#### 버그 수정
- **T키 단축키 오동작 수정**: 검색창·입력란에 타이핑 중 `T` 키를 누르면 다크/라이트 모드가 전환되던 문제 수정 — 입력란 외부에서만 작동하도록 개선

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.1.4.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
