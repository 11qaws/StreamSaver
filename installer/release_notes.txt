## 변경사항

### v1.0.25 — 기록 탭 내보내기 버튼 & 키보드 페이지 이동

#### 새 기능
- **기록 탭 내보내기 버튼**: 설정 탭까지 이동하지 않아도 기록 탭 검색 결과 정보 바에서 바로 JSON/CSV 내보내기
- **방향키 페이지 이동**: 기록 탭에서 ← → 방향키로 페이지를 바로 이동 (입력 필드 포커스 없을 때)

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.0.25.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
