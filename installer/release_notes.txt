## 변경사항

### v1.0.19 — 대시보드 정보 패널 완성

#### 새 기능
- **업데이트 알림 배너**: 새 버전 출시 시 대시보드 상단에 자동 배너 표시, 클릭 시 최신 설치 파일 바로 다운로드
- **다운로드 상세 모달**: 기록 카드 클릭 시 모달 팝업 — 채널, 영상 ID, 업로드 날짜, 재생 시간, 파일 크기, 멤버십 여부, Drive 경로, 파일 경로, YouTube 바로가기 버튼
- **통계 요약 패널**: 기록 탭 상단에 전체 다운로드 수, 총 용량, 멤버십 비율, 채널 수 한눈에 표시
- **`/api/version` 엔드포인트**: 현재 설치 버전 및 저장소 정보 제공

## 설치 방법
1. 아래 `StreamSaver_Setup_v1.0.19.exe` 다운로드
2. 실행 중인 StreamSaver가 있으면 자동 종료 후 업데이트
3. 설치 완료 후 자동 재시작
