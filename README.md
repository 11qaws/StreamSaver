# StreamSaver

## 2. 설치하기

### 2-1. 설치 파일 다운로드

오른쪽 **Releases** 패널 → 최신 버전 → **`StreamSaver_Setup.exe`** 클릭하여 다운로드

### 2-2. 설치 실행

1. 다운로드된 `StreamSaver_Setup_vX.X.X.exe`를 **더블클릭**합니다.
2. "Windows가 PC를 보호했습니다" 창이 나타나면 **추가 정보** → **실행** 클릭
3. "이 앱이 변경을 적용하도록 허용하시겠습니까?" → **예** 클릭
4. 설치 마법사가 열립니다. **다음(Next)** 클릭
5. 추가 옵션 화면:
   - ☑ **Windows 시작 시 자동으로 실행** — 체크 권장 (게릴라 라이브 놓치지 않으려면)
   - □ 바탕화면에 바로가기 만들기 — 원하면 체크
6. **설치(Install)** 클릭 → 1~2분 대기
7. "StreamSaver 지금 실행하기" 체크 후 **완료(Finish)** 클릭

> 화면 오른쪽 아래 시계 옆에 초록 아이콘이 나타나면 설치 성공입니다.  
> 아이콘이 안 보이면 시계 옆 **∧** 버튼을 눌러 숨겨진 아이콘을 확인하세요.

## 3. 처음 연결하기 (최초 1회)

StreamSaver는 Discord 봇과 연결해야 명령을 받을 수 있습니다.  
**한 번만 설정하면 이후에는 자동으로 연결됩니다.**

### Step 1 — Discord에서 페어링 코드 받기

1. Discord를 열고 StreamSaver 봇이 있는 서버로 이동합니다.
2. 채팅창 아무 곳에나 클릭 후 **`/setup`** 입력 → 엔터
3. 봇이 **나에게만 보이는** 메시지로 코드를 보내줍니다.  
   예: `페어링 코드: ABC-123 (10분 유효)`
4. 이 코드를 기억해 두세요. **10분 안에** 사용해야 합니다.

### Step 2 — 트레이에서 코드 입력

1. 화면 오른쪽 아래 StreamSaver 아이콘을 **마우스 오른쪽 버튼**으로 클릭
2. 메뉴에서 **🔗 서버 연결** 클릭
3. 입력창에 Discord에서 받은 코드(예: `ABC-123`) 입력 → **확인**

### Step 3 — 연결 확인

잠시 후 Discord 채널에 "✅ StreamSaver PC가 연결되었습니다." 메시지가 올라오고,  
트레이 아이콘 위에 마우스를 올리면 **"🌐 온라인 모드 — 연결됨"** 이 보이면 완료입니다.

## 4. 영상 다운로드

Discord 채널에서 `/dl` 명령어를 사용합니다.

```
/dl url:https://www.youtube.com/watch?v=XXXXXXXXX
```

YouTube 영상 페이지에서 주소창의 URL을 복사해서 `url:` 뒤에 붙여넣으면 됩니다.

**진행 과정:**
- 봇이 `✅ #1 대기열 추가됨` 메시지를 보냄
- 다운로드 진행 상황이 실시간으로 표시됨
- 완료되면 `✅ #1 완료: [영상 제목]` 메시지가 옴

**저장 위치:** 기본값은 `C:\Users\사용자이름\AppData\Local\StreamSaver\downloads\`  
트레이 아이콘 우클릭 → **📂 다운로드 폴더** 를 클릭하면 바로 열립니다.

> **저장 폴더 변경:** 트레이 우클릭 → **⚙️ 설정 ▶** → **📁 다운로드 폴더 변경** 으로 다른 드라이브나 폴더로 즉시 변경할 수 있습니다. 재시작 없이 바로 적용되며 설정이 영구 저장됩니다.

## 5. 멤버십 영상

멤버십 전용 영상을 다운로드하려면 YouTube 로그인이 필요합니다.

```
/login
```

1. Discord에서 `/login` 입력 → 엔터
2. 잠시 후 **새 Edge 창이 자동으로 열립니다** (평소 쓰는 창과 다른 새 창)
3. 그 창에서 YouTube → Google 계정으로 로그인
4. 로그인 완료 후 Discord에 `✅ 로그인 성공!` 메시지가 오면 완료

> 이후 `/dl`로 멤버십 영상 URL을 입력하면 자동으로 멤버십 쿠키를 사용합니다.  
> 로그인은 약 30일 유지됩니다. 만료 알림이 오면 `/login`을 다시 실행하세요.

## 6. 게릴라 라이브 자동 감지

갑자기 시작되는 라이브 방송을 자동으로 감지해서 녹화합니다.

### 채널 등록

```
/unarchived add url:https://www.youtube.com/@채널핸들 name:표시이름
```

예시:
```
/unarchived add url:https://www.youtube.com/@OuroKronii name:크로니
```

- `url`: YouTube 채널 주소 (`@핸들` 형식 권장)
- `name`: 트레이와 Discord에서 표시할 이름 (자유롭게 설정)

### 기타 명령어

```
/unarchived list                    # 등록된 채널 목록 확인
/unarchived remove name:크로니      # 채널 해제
/unarchived check                   # 지금 즉시 모든 채널 확인
```

> StreamSaver가 실행 중일 때만 감지합니다. PC를 꺼두면 그 시간의 방송은 잡히지 않습니다.

## 7. 트레이 아이콘 안내

### 아이콘 색상 의미

| 색상 | 상태 |
|------|------|
| 초록 | 정상 동작 중 |
| 노랑 | 다운로드 진행 중 |
| 주황 | 경고 (쿠키 만료 임박 등) |
| 보라 | 새 버전 업데이트 있음 |
| 빨강 | 오류 발생 |
| 회색 | 오프라인 (서버 미연결) |

### 우클릭 메뉴 항목 설명

| 항목 | 기능 |
|------|------|
| 🌐 온라인 모드 — 연결됨 | 정상 연결 상태 (클릭 안 됨) |
| 🔄 온라인 모드 — 재연결 중... | 연결이 끊겨 자동으로 재연결 시도 중 (기다리면 됨) |
| 🔌 온라인 모드 — 연결 필요 | 처음 설치 후 아직 연결 안 된 상태 |
| 🤖 1단계: Discord 봇 서버 초대 | 처음 설치 시 봇을 서버에 추가 (봇 미등록 시만 표시) |
| 🔗 2단계: 서버 연결 | 페어링 코드 입력 (봇 미등록 시만 표시) |
| 📊 대시보드 열기 | 웹 브라우저에서 다운로드 기록 보기 |
| 📂 다운로드 폴더 | 저장된 파일 폴더 열기 |
| 📡 Unarchived ▶ | 게릴라 라이브 감시 채널 관리 및 목록 |
| &nbsp;&nbsp;⚙️ 관리 창 열기... | 채널 추가·삭제 다이얼로그 열기 |
| ⚙️ 설정 ▶ | 자동 실행 설정, 가동 시간, 디스크 잔량 등 |
| &nbsp;&nbsp;📁 다운로드 폴더 변경 | 영상 저장 위치를 다른 드라이브로 변경 (즉시 적용) |
| 🔄 재시작 | 연결 문제 시 재시작 |
| ⬆️ vX.X 업데이트 | 새 버전이 있을 때만 표시 — **클릭하면 자동으로 다운로드·설치 후 재시작** |
| ℹ️ StreamSaver vX.X.X | 현재 설치 버전 (클릭 안 됨) |
| ⏹ 종료 | 프로그램 완전 종료 |

## 8. 명령어 목록

| 명령어 | 설명 |
|--------|------|
| `/setup` | 이 채널을 StreamSaver 채널로 설정 + 페어링 코드 발급 |
| `/dl url:주소` | YouTube URL 다운로드 |
| `/waiting` | 진행 중 · 대기 중 목록 보기 |
| `/cancel` | 다운로드 취소 |
| `/status` | PC 연결 상태 확인 |
| `/login` | YouTube 로그인 (멤버십용) |
| `/unarchived add url:주소 name:이름` | 게릴라 라이브 감지 채널 등록 |
| `/unarchived remove name:이름` | 감지 채널 해제 |
| `/unarchived list` | 감지 채널 목록 |
| `/unarchived check` | 즉시 점검 |
| `/help` | 도움말 |

## 9. 자주 묻는 질문

**Q. 트레이 아이콘이 안 보여요.**  
시계 옆 **∧** 버튼 클릭 → StreamSaver 아이콘을 트레이 영역으로 드래그하면 항상 표시됩니다.

**Q. 명령어를 입력해도 봇이 응답하지 않아요.**  
트레이 아이콘 위에 마우스를 올려 "온라인 모드 — 연결됨" 상태인지 확인하세요.  
"접속 시도 중..."이라면 인터넷 연결을 확인하거나 잠시 기다리세요.  
"🔗 서버 연결" 메뉴가 보이면 Discord에서 `/setup`으로 새 코드를 받아 입력하세요.

**Q. 페어링 코드가 만료됐다고 나와요.**  
코드는 10분만 유효합니다. Discord에서 `/setup`을 다시 실행해 새 코드를 받으세요.

**Q. 컴퓨터를 재시작했는데 StreamSaver가 안 켜져요.**  
트레이 우클릭 → **🚀 시작 시 자동 실행** 에 체크하거나,  
바탕화면 바로가기를 더블클릭해서 수동으로 실행하세요.

**Q. 다운로드한 파일이 어디 있나요?**  
트레이 우클릭 → **📂 다운로드 폴더**

**Q. 멤버십 영상이 안 받아져요.**  
`/login`으로 YouTube에 로그인하세요. 로그인이 만료됐을 수 있습니다.

**Q. 업데이트는 어떻게 하나요?**  
새 버전이 나오면 트레이 메뉴에 **⬆️ 업데이트** 항목이 나타나고 알림도 옵니다.  
해당 항목을 클릭하면 자동으로 다운로드가 시작됩니다. 진행률이 표시되고, 완료되면 StreamSaver가 새 버전으로 자동 재시작됩니다. 별도로 설치 파일을 받거나 실행할 필요가 없습니다.
