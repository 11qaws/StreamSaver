📥 StreamSaver 사용 설명서

YouTube 영상 · 라이브 · 멤버십 · 게릴라 방송 자동 저장 프로그램

소개 설치 첫 연결 다운로드 멤버십 게릴라 감지 FAQ

1. StreamSaver란?

StreamSaver는 Discord 채팅창에서 명령어만 입력하면 YouTube 영상을 자동으로 내 PC에 저장해주는 프로그램입니다.


기능설명
🎬 일반 영상 다운로드YouTube 공개 영상을 MP4로 저장
🔴 라이브 방송 녹화진행 중인 라이브도 실시간으로 저장
🔒 멤버십 영상가입한 채널의 멤버십 전용 영상 저장
📡 게릴라 라이브 자동 감지특정 채널에 갑자기 방송이 시작되면 자동 녹화
💡 프로그램은 내 컴퓨터에서만 실행되며, 다운로드 파일도 내 PC에 저장됩니다. 서버 요금이나 클라우드 비용이 없습니다.

2. 설치하기

준비물 확인

항목요구 사항
운영체제Windows 10 / 11 (64비트)
인터넷 브라우저Microsoft Edge (기본 설치됨)
디스크 여유 공간최소 500MB (다운로드 파일 제외)
⚠️ Microsoft Edge는 반드시 설치되어 있어야 합니다.
Edge는 Windows 10/11에 기본으로 설치되어 있습니다. 삭제했다면 microsoft.com/edge 에서 다시 설치하세요.

설치 단계

1
StreamSaver_Setup.exe 파일을 다운로드합니다.

제공받은 설치 파일을 바탕화면이나 다운로드 폴더에 저장하세요.

2
설치 파일을 더블클릭하여 실행합니다.

"이 앱이 변경을 적용하도록 허용하시겠습니까?" 창이 뜨면 예를 클릭하세요.

3
설치 마법사에서 다음(Next)을 클릭합니다.

서버 연결 정보는 기본값 그대로 두고 다음을 클릭하면 됩니다. 변경하지 마세요.

4
"Windows 시작 시 자동으로 실행" 옵션을 선택하는 것을 권장합니다.

컴퓨터를 켤 때 자동으로 StreamSaver가 실행되어 게릴라 방송을 놓치지 않습니다.

5
설치(Install)를 클릭하고 완료될 때까지 기다립니다.

1~2분 정도 소요됩니다.

6
"StreamSaver 지금 실행하기" 체크박스를 선택하고 완료(Finish)를 클릭합니다.

화면 오른쪽 아래 작업표시줄 알림 영역(시계 옆)에 초록색 아이콘이 나타납니다.

💡 아이콘이 보이지 않으면 작업표시줄 오른쪽의 ∧ 버튼을 눌러 숨겨진 아이콘을 확인하세요.

3. 처음 Discord와 연결하기

StreamSaver는 Discord 봇을 통해 명령을 받습니다. 처음 한 번만 연결 설정을 해주면, 이후에는 자동으로 연결됩니다.


1
Discord를 열고, StreamSaver 봇이 있는 서버의 채널로 이동합니다.

봇이 서버에 없다면 서버 관리자에게 추가를 요청하세요.

2
채팅창에 /setup을 입력하고 전송합니다.

봇이 이 채널을 StreamSaver 전용 채널로 설정하고, 페어링 코드를 보내줍니다. (본인에게만 보이는 메시지입니다)

3
봇이 보내준 코드를 기억해 둡니다. (예: ABC-123)

코드는 10분 안에 사용해야 합니다.

4
화면 오른쪽 아래 StreamSaver 아이콘을 마우스 오른쪽 버튼으로 클릭합니다.

메뉴가 나타납니다.

5
🔗 서버 연결을 클릭합니다.

코드 입력창이 나타납니다.

6
봇이 알려준 코드(예: ABC-123)를 입력하고 확인을 클릭합니다.

잠시 후 Discord 채널에 "✅ StreamSaver PC가 연결되었습니다." 메시지가 뜹니다.

7
트레이 아이콘 위에 마우스를 올려 "🌐 온라인 모드 — 릴레이 연결됨"으로 바뀌면 완료!
💡 한 번 연결되면 컴퓨터를 껐다 켜도 자동으로 재연결됩니다. /setup을 다시 할 필요 없습니다.

4. YouTube 영상 다운로드

기본 사용법

Discord 채널에서 다음과 같이 입력합니다:

/dl url:https://www.youtube.com/watch?v=XXXXXXX

YouTube 영상 주소를 복사해서 url: 뒤에 붙여넣으면 됩니다.

다운로드 과정

1
봇이 "✅ #1 대기열 추가됨" 메시지를 보냅니다.
2
다운로드가 시작되면 진행 상황이 표시됩니다.

라이브 방송은 실시간으로 수신 용량이 표시됩니다.

3
완료되면 "✅ #1 완료: [영상 제목]" 메시지가 옵니다.

파일은 내 컴퓨터의 다운로드 폴더에 저장됩니다.

기타 명령어

명령어기능
/waiting현재 진행 중 및 대기 중인 다운로드 목록 보기
/cancel진행 중인 다운로드 취소 (목록에서 선택)
/statusPC 연결 상태 및 다운로드 현황 확인
⚠️ 다운로드 중에는 PC를 끄거나 절전 모드로 두지 마세요. 다운로드가 중단됩니다.

5. 멤버십 영상 다운로드

멤버십 전용 영상을 다운로드하려면 먼저 YouTube 로그인이 필요합니다. 최초 1회만 하면 됩니다.

⚠️ 로그인 전 중요 사항
로그인은 StreamSaver가 별도로 실행하는 브라우저 창에서 진행됩니다. 평소 사용하는 Edge와는 다른 창입니다.
1
Discord에서 /login을 입력합니다.
2
잠시 후 Edge 브라우저 창이 자동으로 열립니다.

새로 뜨는 Edge 창에서 YouTube에 로그인하세요. (본인의 YouTube 계정으로 로그인)

3
로그인 완료 후 Discord에 "✅ 로그인 성공!" 메시지가 옵니다.

이제 멤버십 가입 채널의 영상도 /dl로 다운로드할 수 있습니다.

💡 로그인 정보는 약 30일간 유지됩니다. 만료 알림이 뜨면 /login을 다시 실행하세요.

6. 게릴라(언아카이브) 라이브 자동 감지

특정 YouTube 채널에서 갑자기 라이브가 시작될 때 자동으로 감지하여 녹화합니다. "unarchived"가 제목에 포함된 방송을 주로 잡습니다.

채널 등록하기

/unarchived add url:https://www.youtube.com/@채널이름 name:표시이름

예시:

/unarchived add url:https://www.youtube.com/@OuroKronii name:크로니

등록 명령어 목록

명령어기능
/unarchived add채널 감지 등록
/unarchived remove채널 감지 해제
/unarchived list등록된 채널 목록 보기
/unarchived check지금 즉시 모든 채널 확인
💡 등록된 채널은 5분마다 자동으로 확인합니다. 라이브가 감지되면 자동으로 녹화를 시작하고 Discord에 알림을 보냅니다.
⚠️ 게릴라 라이브 감지는 PC가 켜져 있고 StreamSaver가 실행 중일 때만 작동합니다. PC를 꺼두면 그 시간의 방송은 감지할 수 없습니다.

7. 자주 묻는 질문 (FAQ)

Q. 트레이 아이콘이 보이지 않아요.
작업표시줄 오른쪽의 ∧ (위쪽 화살표) 버튼을 클릭하면 숨겨진 아이콘이 표시됩니다. StreamSaver 아이콘을 드래그하여 트레이 영역으로 꺼내두면 항상 보입니다.
Q. Discord 명령어를 입력해도 봇이 응답하지 않아요.
1. 트레이 아이콘 위에 마우스를 올려 "온라인 모드 — 릴레이 연결됨" 상태인지 확인하세요.
2. 연결이 안 된 상태라면 트레이 우클릭 → 🔗 서버 연결에서 Discord에서 /setup으로 새 코드를 받아 입력하세요.
3. PC가 켜져 있고 인터넷에 연결되어 있는지 확인하세요.
Q. /dl을 입력했는데 "PC가 연결되어 있지 않습니다"라고 나와요.
StreamSaver가 실행 중이지 않거나 서버와 연결이 끊어진 상태입니다.
트레이 아이콘을 찾아 실행 중인지 확인하고, 연결 상태를 확인하세요. 트레이 아이콘이 없다면 바탕화면의 StreamSaver 바로가기를 실행하세요.
Q. 멤버십 영상이 다운로드가 안 돼요.
먼저 /login으로 YouTube 로그인을 해야 합니다.
로그인 후에도 안 된다면 로그인이 만료된 것입니다. /login을 다시 실행하세요.
Q. 다운로드한 파일이 어디에 있나요?
트레이 아이콘 우클릭 → 📂 다운로드 폴더를 클릭하면 파일이 저장된 폴더가 열립니다.
Q. StreamSaver를 종료하고 싶어요.
트레이 아이콘 우클릭 → ⏹ 종료를 클릭하세요.
단, 종료하면 게릴라 라이브 감지도 중단됩니다.
Q. 컴퓨터를 재시작했는데 StreamSaver가 안 켜져 있어요.
설치 시 "Windows 시작 시 자동으로 실행" 옵션을 선택하지 않으셨다면 수동으로 실행해야 합니다.
트레이 아이콘 우클릭 → 🚀 시작 시 자동 실행에 체크하면 앞으로는 자동으로 켜집니다.
Q. 페어링 코드가 만료되었다고 나와요.
코드는 10분간 유효합니다. Discord에서 /setup을 다시 실행해서 새 코드를 받고, 트레이 우클릭 → 🔗 서버 연결에서 입력하세요.
